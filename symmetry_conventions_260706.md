## 2. 이론: 단일 가정 α = 2 (유효 여자 전류 2배)

이 스크립트 계열의 대칭 모델은 각 도체 조각의 절단면 양쪽에 터미널을 2개씩
(in=Positive, out=Negative) 걸고 둘 다 같은 winding에 넣는다. 이 이중 터미널 관례가
대칭 경계의 이미지 전류와 결합해 **유효 여자가 정확히 2I로 작용**한다 (α=2).
이 하나의 가정에서 모든 관측이 유도된다:

| 물리량 | 유도 | 예측 | 실측 |
|---|---|---|---|
| 전류밀도 | J' = αJ | 2× | (손실에서 역산 2.0×) |
| 손실밀도 | p' = ρJ'² = α²p | **4×** | 4.0× (적분/체적으로 환산) |
| 오브젝트 손실적분 | p'×(체적/2^c) = P×4/2^c | 4/2^c | §1 표와 일치 |
| B (선형) | B' = αB | 2× | (모델 3종 audit로 직접 측정) |
| 자기에너지(영역) | W' = α²W/8 = W/2 | — | — |
| 인덕턴스 | L' = 2W'/I² = (α²/8)L = **L/2** | 0.5× | 0.50002× |
| 결합계수 | k' = M'/√(L'L') = k | 1× | 1.0003× |
| 코어손실 | ∝(αB)^y = 2^1.74 = 3.34× (체적분율 곱해짐) | 비정수 배 | §4 참고 |

L의 α²/8 유도: 대칭 영역 에너지 = (밀도 α²) × (체적 1/8) = α²W/8, L = 2W'/I²_명목.
α=2에서 정확히 1/2 — 실측 1.9997~2.0021과 소수점 셋째 자리까지 일치.

## 3. Icepak AssignEMLoss Loss Multiplier

Icepak의 EM Loss 링크는 Maxwell 해의 **손실밀도 분포**를 매핑한다. 대칭 해의
밀도는 실제의 α² = 4배이므로:

> **Loss Multiplier = 1/α² = 0.25**

- Icepak 지오메트리가 풀이든 대칭 조각이든 동일하게 0.25 (밀도가 4배인 것이므로).
- 검증법: 매핑 후 Icepak이 표시하는 "imported total power"를 알려진 손실값과 대조.
- 이 0.25는 **본 스크립트 계열의 이중 터미널 전류 여자 방식에 특유**한 값이다.
  단일 터미널 관례의 반/쿼터 모델은 다른 계수를 가지므로 타 모델에 일반화 금지.

## 4. 대칭 모델에서 코어손실이 보정 불가능한 이유

코어손실 밀도 ∝ B^y (y=1.74, 비선형). 대칭 해의 B가 2배이므로 손실밀도는
2^1.74 = 3.34배 — 전기 손실(4배)과 다르고, 체적분율(2^-c)과 결합하면 오브젝트마다
3.34/2^c 배가 되어 **하나의 multiplier로 복원 불가**. 실측: 무부하 mini에서 대칭
코어손실 64.8W vs 풀 46.5W (비 1.39 = 3.34/2 와 3.34의 체적 가중 혼합, 예측 부합).

→ 따라서 코어손실이 필요한 해석(design2 maxwell_loss)은 **풀모델로 강제**되어 있고,
   열해석 손실 주입도 풀모델 값을 그대로 사용한다 (multiplier 불필요).

## 6. 실무 요약

- **L/k/누설**: 대칭 매트릭스 값 ×2 = 실제 (k는 그대로). 누설도 이론상 ×2이나
  수치 민감도 ±10% — 정밀 판정은 풀모델.
- **권선/플레이트 손실**: 대칭 값은 오브젝트별 4/2^c 관례 — 헷갈리면 풀모델 loss 디자인 사용.
- **코어손실·자속밀도**: 대칭 모델 사용 금지 (비선형). 풀모델 전압원 여자로 원샷.
- **Icepak EM Loss 링크**: 대칭 Maxwell 해에서 import 시 multiplier 0.25.
  (새 파이프라인은 풀모델 + 숫자 주입이라 해당 없음)
